Available Commands

/discover [search term]

Search and discover OWASP projects. Use without arguments to see OWASP-BLT repositories, or add a search term to find projects by name, technology, or topic.

Example: /discover python or /discover security

/blt [subcommand]

Multi-purpose command with several subcommands:

  • user <username> - View OWASP profile for a GitHub user
  • chapters [location] - Browse OWASP chapters worldwide
  • projects [search] - Discover OWASP projects
  • gsoc [search] - View GSoC 2025 projects
  • events [category] - Get upcoming OWASP events
  • committees [name] - View OWASP committees
/gsoc25 [search]

Search Google Summer of Code 2025 projects. Filter by technology, project name, or mentor.

Example: /gsoc25 python or /gsoc25 mentor:donnie

/stats

Get comprehensive statistics about the OWASP BLT platform including project counts, user statistics, domains, hunts, and more.

/contrib

Learn how to contribute to OWASP projects. Get guidance on joining project channels, finding projects, and submitting contributions.

/help

Display a list of all available commands with brief descriptions.

/report <description>

Report a bug or issue directly from Slack. Your report will be logged and tracked in the system.

/apps

View a list of all Slack apps installed in your workspace.
