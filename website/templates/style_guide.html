BLT Style Guide

Design System Overview

Our design system ensures consistency across the BLT platform, creating a cohesive and professional user experience. This guide serves as the single source of truth for all design decisions and component implementations.

Key Principles

  • Consistency: Maintain visual and functional consistency across all pages
  • Accessibility: Ensure all components are accessible to all users
  • Responsiveness: Design for all screen sizes and devices
  • Performance: Optimize for fast loading and smooth interactions

Icons

Icon System

BLT uses a consistent icon system based on Heroicons for a cohesive visual language across the platform. Icons should be semantic and used to enhance user experience, not just for decoration.

Icon Guidelines

  • Maintain consistent sizing within context groups
  • Use the primary color (#e74c3c) for interactive icon elements
  • Ensure sufficient touch target size (44x44px) for interactive icons
  • Include appropriate labels or tooltips for accessibility
  • Consider using outlined icons for UI elements, solid for active states

Responsive Design

Breakpoints

  • sm: 640px - Mobile landscape
  • md: 768px - Tablets
  • lg: 1024px - Laptops
  • xl: 1280px - Desktop
  • 2xl: 1536px - Large screens

Responsive Guidelines

  • Use mobile-first approach in all designs
  • Implement responsive grids using grid-cols-1 md:grid-cols-2
  • Adjust font sizes for different screen sizes
  • Ensure touch targets are at least 44x44px on mobile
  • Test layouts across all breakpoints

Accessibility

Color Contrast

  • All interactive elements must have sufficient color contrast
  • Text should be readable at all sizes
  • Use WCAG 2.1 AA standards for contrast ratios

Interactive Elements

  • Interactive elements should have clear focus states
  • Include hover and active states
  • Ensure keyboard navigation works

Content Structure

  • Use semantic HTML elements
  • Include proper heading hierarchy
  • Images should have appropriate alt text
  • Forms should have associated labels
