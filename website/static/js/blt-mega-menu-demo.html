BLT Mega Menu — Demo Page

The OWASP BLT mega menu bar appears at the top of this page. It is injected entirely by the script above — no other HTML or CSS required.

To use on any site, drop in one line:

<script src="https://blt.owasp.org/static/js/blt-mega-menu.js"></script>

The script is fully self-contained. It injects its own styles and HTML, requires no dependencies, and works in any modern browser.
