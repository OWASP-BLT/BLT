# Legal Documentation

This section contains legal documents and policies for the OWASP BLT platform.

## Documents

- [Terms of Service](terms-of-service.md) - Terms and conditions for using BLT

## Key Points

- BLT is 100% free to use, Open Source, and a non-commercial, not-for-profit initiative
- All rewards go directly to security researchers who find and report vulnerabilities
- We are committed to transparency and protecting user privacy
