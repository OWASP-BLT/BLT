# BLT Action Items

Action items captured from community discussion thread.

## Leaf / PR Review Extension

- [ ] **Flakiness detection** — Detect flaky tests in CI runs and surface generic guidance (e.g. "this test is historically flaky — retry before investigating").
- [ ] **Duplicate work detection** — Flag PRs that overlap in scope with other open PRs or recently merged work.
- [ ] **Automatic flaky-test flagging** — When CI fails due to a known-flaky test, flag it proactively without requiring contributors to explicitly ask an AI agent.
- [ ] **PR progress tracker** — Show how much of a PR's stated scope has been implemented, what is in-scope vs. out-of-scope, and what remains.
- [ ] **Mass AI-generated PR detection** — Identify and flag PRs that appear to be bulk-generated by AI without meaningful human review or customisation.
- [ ] **Minimal AI feature** — First working AI integration for Leaf (in progress — see upcoming PR from @DonnieBLT).
